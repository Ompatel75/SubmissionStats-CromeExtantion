# CF Submission Stats

A Chrome extension that analyzes Codeforces submissions and shows:

- Runtime percentile
- Memory percentile
- Best / Average / Median stats
- Comparison with all accepted submissions

## Features

- Beautiful UI
- Fast API fetching
- Progress loader
- Supports:
  - Contest problems
  - Problemset problems
  - Gym problems

## Installation

1. Download this repository
2. Open Chrome
3. Go to:
   chrome://extensions/
4. Enable Developer Mode
5. Click "Load unpacked"
6. Select this folder

## Tech Stack

- JavaScript
- CSS
- Chrome Extension Manifest V3
